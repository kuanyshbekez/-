# Bedrock Bot — стартовый проект (Minecraft Bedrock 1.21+)

Коротко:
- Этот бот использует `bedrock-protocol` и предназначен для Bedrock серверов без Xbox (offline/no auth).
- Базовые возможности: подключение к серверу, простые команды через чат (!follow, !stop, !collect), попытка перемещения/следования за игроком и простая логика поиска ближайших чанков.

Требования:
- Node.js 18+
- GitHub Codespaces или любой VPS/ПК
- Сервер Bedrock 1.21+ (offline/no xbox)

Установка и запуск:
1. Клонируйте репозиторий в Codespace или на машину.
2. Установите зависимости:
   npm install
3. Откройте config.json и при необходимости измените username/targetPlayer/поведение.
   (Адрес по умолчанию: Autysty.aternos.me:28834 — уже подставлен)
4. Запустите:
   npm start
   или
   node bot.js

Команды (в чате сервера):
- !follow <игрок> — бот начнёт следовать за <игрок>.
- !stop — бот остановится.
- !collect <block_name> — попытаться собрать ближайший блок указанного типа (например: !collect oak_log).

Ограничения и примечания:
- Bedrock-протокол и события различаются в зависимости от сервера. Этот бот — стартовая база, многие события/инвентарь/крафт требуют расширения.
- Сервер должен позволять подключение без Xbox-аутентификации (вы указали, что так и есть).
- На серверах с анти‑читом бот может кикнуться. Для продвинутых задач — дополнительно настроим плавность движений, задержки и логику имитации.
